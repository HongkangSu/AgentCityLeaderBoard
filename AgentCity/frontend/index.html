LibCity Agent

Monitor the Claude-driven automation that scouts papers, ports models into LibCity, validates runs, and reports metrics—styled in the spirit of the LibCity landing page.

通过自动化代理完成最新论文检索、模型迁移与验证，全流程透明可追踪。

Autonomous Workflow 2025 Editions: ICLR · ICML · NeurIPS · KDD · ICDE

文献搜索 & 下载 / Literature Search & Downloads

输入关键词即可筛选最新的交通预测论文，同时可直接访问 PDF；Use the filters below to scope by year and conference.

模型迁移批处理 / Migration Batch Runner

在下载的论文中勾选目标，导出迁移计划，先运行迁移再单独触发调参；Select saved papers with code, run migration first, then launch tuning separately.

选择论文 / Select Papers

仅展示已成功保存 PDF 的论文。Only papers with archived PDFs & repos appear here.

步骤 1：迁移 / Step 1: Migration

生成的 JSON 包含仓库与模型，可直接交由管线执行；Generated JSON is ready for python claude_client.py

步骤 2：调参 / Step 2: Tuning

迁移结束后再运行此步骤，复用所选论文进行超参搜索。Run this after migration to tune hyperparameters.

任务队列 / Job Queue

实时关注文献搜索、模型迁移与调参的运行状态。Track live status of literature scans, migrations, and tuning runs.

Stage Timeline / 阶段时间线

Each block reflects a pipeline stage and its most recent transcript. 每个条目展示阶段概览与片段。

Archived Articles / 论文归档

Saved PDFs and metadata gathered by the catalog_article tool. 所有下载记录与抓取到的元信息可在此浏览。
