## Special Requirements

### 1. Dependencies
```bash
# Required for GAT graph encoder
pip install torch-geometric

# If not available, set mode='p' in config for plain embeddings
```

### 2. Data Format
- **Ideal**: GPS trajectories + road network graph
- **Compatible**: POI check-ins (handled by JGRMEncoder)
- **Features**: 8D GPS features (synthetic or real)

### 3. Computational Requirements
- **GPU Memory**: ~4GB (default), ~8GB (large config)
- **Training Time**: ~2-15 hours depending on dataset size
- **Queue Buffer**: 2048 * hidden_size for contrastive learning

## Dataset Requirements

### Standard Trajectory Data (Minimum)
- Location sequences (POI IDs or road segment IDs)
- Timestamps for each check-in
- User information

### Ideal GPS Trajectory Data
- GPS point sequences with coordinates
- Road network topology (adjacency matrix)
- Map-matched trajectories
- Rich features: speed, acceleration, heading

### Encoder Handles
The JGRMEncoder automatically:
- Extracts temporal features (week, minute, intervals)
- Generates synthetic GPS features from POI data
- Constructs graph from trajectory co-occurrence
- Creates dual-stream format (GPS + Route)

---

## Pre-training Objectives

### 1. Masked Language Modeling (MLM)
- Predict masked road segments from GPS stream
- Predict masked road segments from Route stream
- Loss weight: 1.0 each

### 2. GPS-Route Matching
- Contrastive learning with queue (2048 negatives)
- Temperature-scaled similarity (tau=0.07)
- Loss weight: 2.0

### Combined Loss Formula
```python
total_loss = (
    mlm_loss_weight * gps_mlm_loss +
    mlm_loss_weight * route_mlm_loss +
    match_loss_weight * matching_loss
) / (2 * mlm_loss_weight + match_loss_weight)
```

---

## Known Limitations

1. **Data Complexity**: Standard POI datasets lack real GPS features
2. **Graph Requirement**: Needs road network (constructed from co-occurrence if missing)
3. **Memory Usage**: Queue-based contrastive learning requires additional memory
4. **Computation**: Dual-stream processing increases computational cost
